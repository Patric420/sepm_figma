# Example Discovery Call Transcript: SaaS Dashboard Project

**Participants:**
- **Elena (Consultant):** Lead Architect at NexusDevs
- **David (Client):** Product Manager at FinStream Solutions

---

**Elena:** Hi David, thanks for joining today. I've had a chance to look at the high-level brief you sent over for the FinStream dashboard. To get a solid Statement of Work together, I'd love to dive into the specifics of the scope and timeline.

**David:** Absolutely. So, the core issue is our legacy dashboard is slow and built on an old jQuery stack. We want to move everything to React. 

**Elena:** Makes sense. Are we talking about a complete visual overhaul as well, or just a technical migration?

**David:** Both. We need a modern, "glassmorphism" style dark mode. It needs to feel more like a premium fintech tool. Functional-wise, we need three main views: an Analytics Overview with real-time charts, a Transaction History table with advanced filtering, and a User Permissions management pane.

**Elena:** Got it. For the Analytics, will we be using your existing REST API, or do we need to build new endpoints for the real-time data?

**David:** The existing API handles most things, but we’ll need new endpoints for the weekly growth projections. Our team can handle the backend if you provide the specs, but we’d prefer if you handled the full integration.

**Elena:** We can certainly handle the integration. Now, let's talk timeline. You mentioned "end of Q2" in your email. Is that a hard deadline for the full launch?

**David:** Yes, we have a major investor showcase in the first week of July. We need at least a stable Beta by June 15th for internal testing.

**Elena:** That gives us about 10 weeks. We can structure this into three phases: 
1. UI/UX Design & Prototyping (Weeks 1-3)
2. Core Component Development & API Integration (Weeks 4-8)
3. Testing, Polishing, and Beta Release (Weeks 9-10)

**David:** That sounds reasonable. What about the budget? We were hoping to keep this first phase under $45,000.

**Elena:** Based on the complexity of the filtered tables and real-time charts, I think $42,500 is a fair estimate for this scope. This would include the React frontend, the new API integration, and the full design system.

**David:** Okay, $42.5k works for us. Can you get me a formal SOW by Friday?

**Elena:** I'll do you one better—I'll use our new tool to generate a draft from this transcript right now and send it over for your review.

**David:** Perfect. Looking forward to it.

**[End of Transcript]**
